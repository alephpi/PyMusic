自学过程中碰到发现纯律里面的三全音（tritone）有好多不同的版本：7/5(Huygens),10/7(Euler),11/8,45/32,64/45...
这些文章交代了这种现象，原因是纯律的定调五花八门（不像平均律或五度相生律那样唯一）。但具体没有介绍不同定调方法，所以看得仍然云里雾里。
